# 7 Powers Analysis (Hamilton Helmer)

Apply Helmer's framework to evaluate which durable competitive moats (Scale Economies, Network Effects, Switching Costs, Branding, Cornered Resource, Counter-Positioning, Process Power) apply or could be built.

## Grok Usage

Delegate large analyses to the `strategy-analyst` persona using the task tool.

## Process

1. Load company strategy context.
2. For each of the 7 powers, assess current strength (Strong / Emerging / Absent / Buildable).
3. Identify the one or two powers the company is best positioned to strengthen.
4. Deliver clear implications for product roadmap, GTM, and resource allocation.

Save key assessments to memory and context.
