# PM Copilot — Grok Plugin

This is the official Grok plugin distribution of PM Copilot.

## Installation

The recommended way to install this plugin is through the Grok plugin system (once published) or by copying this folder into your Grok plugins directory.

## What's Included

- 65+ PM skills
- 8 specialized PM personas / sub-agents
- Strong `AGENTS.md` with session protocols and quality bar
- Context and memory scaffolding

## Primary Features on Grok

- Excellent integration with Grok's native subagent + persona system
- Designed to work well with Grok's native memory
- Clean slash command support (`/write-prd`, `/discover`, etc.)

See the main project README for full usage instructions.
